## React-Carousel-Component
This component will set carousel page and indicators quantity with parameters. You just need a response format like CarouselContents.json file. If you get a response which has another format, you just need to change some lines in component. There is a static sample background image used for easy understanding to component but if we want to use various images on every slides of carousel, we can modify component like an image url comes with every response data items.
